# PinBall
by Gavin McGuire
for UNM CS351

## How to play

* Click on jar file to start game.
  

* You can move the ball around by just moving in on the window.
* Clink on the window or "play" / launch the ball.
* Click "reset" button to reset ball, board, and score.
